# 喵搭 达人端 APP

基于 React Native 0.75+ 的上门服务O2O平台达人端。

## 技术栈

- React Native 0.75.4
- React Navigation 6.x
- Zustand 4.5+
- TanStack Query 5.x
- Phosphor Icons
- 高德地图（实时定位、导航）

## 核心功能

- 工作台：今日数据、收入统计、快捷入口
- 订单管理：待接订单、当前订单、历史订单
- 服务流程：接单 → 出发 → 到达 → 服务 → 完成
- 实时定位：后台持续上报位置
- 收入管理：收入统计、提现申请
- 个人中心：资料管理、设置

## 启动

```bash
npm install
npm start
npm run android  # 或 npm run ios
```
